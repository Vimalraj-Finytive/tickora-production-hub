# Tickora Professional Dashboard

## How to Run

1. Ensure you have Python installed.
2. Run `pip install plotly docker psutil pandas streamlit` (if not already done).
3. Double-click **`run_dashboard.bat`** to start the dashboard.

## Features

- **Real-Time System Metrics**: CPU, RAM, Disk, Uptime.
- **Docker Monitoring**: Live container status and health checks.
- **Live Graphs**: Interactive Plotly area charts for historical data.
- **Professional UI**: Dark mode with "Grafana-style" aesthetics.
